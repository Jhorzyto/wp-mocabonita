\MocaBonita

\MocaBonitaMocaBonita

Um framework para o desenvolvimento de plugins na plataforma wordpress.

O padrão singleton é útil quando precisamos ter certeza de que só temos uma única instância de uma classe para todo o ciclo de vida do pedido em um aplicativo da Web. Isso normalmente ocorre quando temos objetos globais (como uma classe de configuração) ou um recurso compartilhado (como uma fila de eventos).
