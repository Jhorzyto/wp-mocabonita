indexAction()

indexAction() : null|string|\MocaBonita\view\View|void

Ação principal da controller

Se o retorno for null, ele irá chamar a view desta controller e redenrizar Se o retorno for string, ele irá imprimir a string na tela Se o retorno for View, ele irá redenrizar a view desta controller Se o retorno for qualquer outro tipo, ele irá fazer um var_dump do retorno
